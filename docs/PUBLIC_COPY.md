# Public Copy

## Copy Standard

Public wording should describe the app as an educational measurement-uncertainty tool. It should
not claim diagnostic authority, treatment guidance, regulatory clearance, or clinical validation.

## Required Themes

- Measurement uncertainty, not medical advice.
- Client-side calculation and no intentional storage of patient values.
- Default variability values are documented estimates, not local laboratory calibration.
- Local clinical judgment and local laboratory performance data take precedence.

## Review Triggers

Use `.agents/skills/public-copy-safety-review/SKILL.md` for README, web UI, footer, release note,
and repository-description edits that affect user expectations.
