# Clinical Scope

## Intended Use

This project is an educational measurement-uncertainty aid for thinking about changes between
two sodium results. It is meant to make analytic and preanalytic uncertainty more visible.

## Non-Goals

- It is not medical advice.
- It is not a diagnostic or treatment recommendation engine.
- It is not a medical device.
- It is not an EMR integration.
- It does not store, transmit, or persist patient data by default.

## Scope

The app models uncertainty around observed sodium values using documented measurement-error
parameters. It does not infer the cause of sodium change, correct for specimen artifacts, or
replace local laboratory quality data.

## Limitations

Published limits of agreement and default variability estimates may not match local instruments,
collection practices, patient populations, or calibration states. Users should treat outputs as
measurement-uncertainty intuition, not as a clinical decision rule.
